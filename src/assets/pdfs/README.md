# PDF Assets Folder

This folder contains downloadable PDF files for lead magnet forms.

## Organization

Place PDFs directly in this folder or organize by type (reports, guides, case-studies, etc.)

## Usage

Local files: Use `/assets/pdfs/filename.pdf` in page front matter
External URLs: Use full URL in page front matter

## Integration

PDFs are downloaded instantly on form submission.
N8N workflow also sends email with download link.
